Agrega un botón "Imprimir Álbum MSDS" en la página msds.tsx, visible arriba del buscador. Al hacer clic debe abrir una ventana de impresión con todas los productos que tienen MSDS del almacén actualmente seleccionado, organizados en un álbum de 6 tarjetas por hoja A4.
Datos a usar:
Filtra de la lista de productos ya cargada los que tengan msds: true y msdsUrl con valor. Pásalos todos a la ventana de impresión como JSON serializado en el HTML.

Cada 6 tarjetas se inserta un salto de página:

Agrupa los productos en chunks de 6 con JS y cierra cada grupo con un <div class="page-break">.
Encabezado por hoja — repite en cada página un header con:

Con texto izquierdo: "Álbum MSDS — <nombre del almacén>" en bold 13px, y texto derecho: fecha actual en 11px gris. Usa position: running(header) si el navegador lo soporta, o simplemente repite el div de header al inicio de cada grupo de 6.
Pie de página por hoja:
css.footer { font-size: 9px; color: #aaa; text-align: center; margin-top: 8px; }
Texto: "Documento generado el <fecha> — Solo para uso interno de emergencia"
Script de inicialización:
js// Cargar JsBarcode desde CDN
// Esperar a que cargue antes de imprimir
const script = document.createElement('script');
script.src = 'https://cdn.jsdelivr.net/npm/jsbarcode@3.11.5/dist/JsBarcode.all.min.js';
script.onload = function() {
  document.querySelectorAll('.barcode').forEach(el => {
    JsBarcode(el, el.dataset.code, {
      format: 'CODE128',
      displayValue: false,
      height: 30,
      margin: 0,
      width: 1.2
    });
  });
  window.print();
};
document.head.appendChild(script);
Cada <svg class="barcode" data-code="${product.code}"> se inicializa con ese script.
